# Persi Stock

Um sisteminha de estoque que eu fiz em Python pra treinar.

## O que ele faz?

- Cadastra produtos (nome, quantidade, preço)
- Remove produtos
- Mostra tudo que tá no estoque

## Como usar?

1. Baixa o código
2. Abre o terminal na pasta
3. Digita python main.py
4. Escolhe as opções do menu

## O que eu aprendi fazendo isso?

- Usar as listas e dicionários juntos
- Fazer um menu que fica repetindo até a pessoa escolher sair
- Validar se o usuário digitou número negativo
- Subir projeto pro GitHub (essa parte foi tensa)
