# Persi Stock

Um sisteminha de estoque que eu fiz em Python pra treinar.

## O que ele faz?

- Cadastra produtos (nome, quantidade, preço)
- Remove produtos
- Mostra tudo que tá no estoque

## Como usar?

1. Baixa o código
2. Abre o terminal na pasta
3. Digita `python main.py`
4. Escolhe as opções do menu
